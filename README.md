# Java Master AI ☕🤖

## Overview

Java Master AI is an AI-powered Java learning application developed using Python, Streamlit, and the Google Gemini API. It helps students learn Java programming by answering coding questions, explaining Java code, debugging errors, and providing theory explanations through an interactive AI interface.

This project was built as an internship project to demonstrate the practical use of Generative AI in programming education.

---

## Features

* 💻 Ask Java Coding Questions
* 📖 Explain Java Code
* 🐞 Debug Java Code
* 📚 Java Theory Question Answering
* ⚠️ Error Handling
* 🧩 Modular Python Project Structure
* 🤖 AI-Powered Responses using Google Gemini
* 🎨 Interactive Streamlit User Interface

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/zidsscode/Java-Master-AI.git
```

### 2. Open the project

```bash
cd Java-Master-AI
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file and add your Gemini API key

```env
GEMINI_API_KEY=your_api_key_here
```

### 5. Run the application

```bash
streamlit run app.py
```
